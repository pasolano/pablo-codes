μSE is a mutation-based soundness evaluation framework which systematically evaluates Android static analysis tools to discover, document, and fix flaws.

I contributed to this project while I was enrolled in W&M's Software Engineering course, CSCI 435, Fall 2020. My contributions to this project were mostly related setting up and maintaining the integrity of μSE's CI/CD.

Code belongs to a private repository, but samples may be provided by request.
